1. 오늘 목표

- pull to refresh 구현

- 코어 데이터 검색 기능 구현

2. 오늘 성과

- pull to refresh을 구현하여 페이지 내에서 새로고침을 할 수 있도록 했습니다.

- 코어 데이터 검색 기능 구현하여 메일 검색이 가능하도록 했습니다.



3. 오늘 느낀 점/어려운점

- 추가하는 것이 간단한 것 같으면서도 생각할 점이 많았습니다. 이 기능들을 추가함으로써 데이터를 동기화하는 과정에서 데이터가 더 추가되거나 삭제될 수 있었고, 이를 해결하기 위해 검색바가 나타나는 타이밍을 조정했습니다. pull to refresh를 구현하며 저도 모르게 많은 줄의 코드를 작성하고 있었고, 이 시간을 통해 코드를 조금이나마 정리하는 시간을 가졌습니다.
